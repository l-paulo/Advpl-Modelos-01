# Ferramentas úteis para utilização com ADVPL

## Manipulação de Array

[aMap](./Arrays/aMap/) -  O método map() invoca a função callback passada por argumento para cada elemento do Array e devolve um novo Array como resultado.

[aFilter](./Arrays/aFilter/) - O método filter() cria um novo array com todos os elementos que passaram no teste implementado pela função fornecida.

[ValArray](./Arrays/ValArray.prw/) - Recuperar informação dentro de um Array, validando se existem as posições passadas.
